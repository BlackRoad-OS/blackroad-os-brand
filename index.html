🔐 Identity Format

All BlackRoad identities use the br1_ prefix followed by a nanoid.

Format Examples

br1_abc123xyz456 // Agent identity
ar_xyz789def012 // Agency response ID
evt_mno345pqr678 // Ledger event ID
agent_stu901vwx234 // Agent record ID
