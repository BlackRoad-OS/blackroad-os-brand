# Assets

This directory contains various design and media assets for BlackRoad OS.

## What Belongs Here

- **Images**: Photography, illustrations, and graphics used in marketing materials
- **Videos**: Promotional videos, product demos, and tutorials
- **Audio**: Sound effects, jingles, or podcast intro/outro music
- **Downloadable resources**: Assets that can be shared with the community
- **Social media assets**: Profile pictures, cover images, and post templates
- **Mockups**: Device mockups and screenshot templates

## File Organization

Organize assets by type and purpose:
- `/images` - Photos, illustrations, graphics
  - `/photos` - Photography assets
  - `/illustrations` - Vector illustrations
  - `/graphics` - General graphic assets
- `/videos` - Video content
- `/social-media` - Social platform-specific assets
- `/mockups` - Device and UI mockups
- `/downloads` - Community-shareable resources

## File Naming Conventions

Use clear, descriptive names:
- Use lowercase with hyphens: `blackroad-logo-hero.png`
- Include dimensions for images: `banner-1920x1080.jpg`
- Version files when needed: `screenshot-v2.png`
- Use appropriate file extensions

## Supported Formats

### Images
- PNG (with transparency)
- JPG (for photos)
- SVG (for scalable graphics)
- WebP (for web optimization)

### Videos
- MP4 (H.264/H.265)
- WebM (for web)
- MOV (for editing)

### Audio
- MP3 (compressed)
- WAV (uncompressed)

## Usage Guidelines

- Ensure all assets follow brand guidelines from `/brand-kit`
- Optimize file sizes for web delivery
- Include source files (PSD, AI, Figma links) when possible
- Document licensing and attribution requirements
- Refer to `/guides/usage-rules.md` for usage restrictions
