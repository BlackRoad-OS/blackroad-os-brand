Paragraphs

Body · Small · Meta

Body copy

Standard body

BlackRoad OS treats every project as a road. The typography underneath is quiet: white and soft grey on pure black, with a comfortable line length so you can actually read what the interface is saying.

Body with link

When we point to a surface, we link once and move on. Example: view the architecture map instead of pasting the full URL into the copy.

Small & meta

Small helper

Use small text for helper lines under inputs, short disclaimers, and light guidance in cards. If it feels like a whisper, the size is probably right.
