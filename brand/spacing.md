# Spacing & Layout

BlackRoad OS spacing uses a 4px base rhythm to keep UI, decks, and assets aligned. Combine tokens to create predictable patterns and generous breathing room.

## Scale

```
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128
```

## Usage

- Default gutters and padding: 16–24px on desktop; 12–16px on mobile.
- Card padding: 24px on dark surfaces; increase to 32px for hero tiles.
- Section spacing: 64–96px between major bands on pages or presentations.
- Corner radii: 8–12px for UI components to preserve the BlackRoad OS feel.

## Layout Notes

- Align to a 4px grid; avoid odd pixel values that break the neon geometry.
- Keep negative space around logos and primary statements; never crowd the insignia.
- When pairing gradients with content, maintain a minimum 24px inset to avoid glow bleed.
