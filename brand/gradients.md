# Gradients

Gradients capture the Neon Road horizon. Apply them sparingly to emphasize flow, signal transitions, or frame hero content.

## Gradient Tokens

| Token | Value | Use |
| --- | --- | --- |
| `--gradient-br-os` | `linear-gradient(90deg, #ff4fd8, #4ffff2)` | Primary sweep for hero bars, headers, and call-to-action rails. |
| `--gradient-road-horizon` | `radial-gradient(circle, #4ffff2, #000000)` | Vignette glow behind logos, SIG spirals, or key stats. |
| `--gradient-sig-spiral` | `conic-gradient(from 45deg, #ff4fd8, #b44bff, #4ffff2)` | Dynamic rings, data arcs, and fractal-road motif accents. |

## Usage Guidance

- Keep gradients aligned to components; avoid random directions.
- Pair with `--br-black`/`--br-surface` backgrounds to avoid washout.
- Limit to one major gradient per viewport to maintain clarity.
- Add subtle blur/glow only when it reinforces hierarchy or motion.
